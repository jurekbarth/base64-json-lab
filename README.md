# 64 Lab

Base64 encoder and decoder for text and JSON.

Live app: [64.v10.app](https://64.v10.app)

## Features

- Decode Base64 to text
- Decode Base64 to formatted JSON
- Encode text to Base64
- Encode JSON to Base64

## Local

Open `index.html` in a browser.

## Deploy

The project is configured for Vercel and GitHub Actions.
